This Shiny R application can be run locally by loading "server.R" and clicking "Run App" in R studio, or using runApp() in the "shiny" package

This R shiny application is also available at: https://morganpkain.shinyapps.io/twomey_et_al_color_mimicry/

This shiny app supports "default" skin pigments (used for the results in the main manuscript), and user defined pigments. To define pigments, fill in "rgb_pigment_user_def.csv" and "pigment_reflectance_user_def.csv" in the same format as "rgb_pigment.csv" and "pigment_reflectance.csv". To choose these pigments, select the "Yes" radial button for "User Defined Pigments?:"
